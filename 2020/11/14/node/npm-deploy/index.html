【npm】利用npm安装/删除/发布/更新/撤销发布包

  • npm adduser用于在npmjs.com注册一个用户。
  • 已注册 npm login
  • npm publish

注意：登录时报错：e409 Conflict

出现这个错误最大的可能性是我们把registry选项替换成了淘宝的镜像

登录前确保npm源为官方源：

解决方案

细节：

  • 发布一个beta版，默认为latest

  • 如果想废弃某个版本的模块，可以使用npm deprecate命令。

  • 模块的维护者可以发布新版本。npm owner命令用于管理模块的维护者。
